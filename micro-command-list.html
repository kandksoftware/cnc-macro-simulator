Micro command list

Micro Commands is a syntax sugar for the ISO (RS-274) standard, extending it with useful cycles and functions that greatly speed up and facilitate the processing of more complex cnc programs,

Good to know

Micro commands are not part of the RS-274 standard, so programs created using them must be exported to pure g code understandable to most cnc machines.
