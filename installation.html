Installation / reinstallation

CNC Macro Simulator II was to be available from the very beginning on many platforms, such as Windows, MacOS, Linux, Android and IOS. Therefore, it is distributed as PWA by default. Although in this form, the software takes up much less space on the computer's disk, having practically the same advantages as any other native software. Unfortunately, that also has a downside: Google Chrome or Microsoft Edge is required to install CNC Macro Simulator II.

Good to know

However, if you prefer a standard file (e.g. .exe for Windows or .app for Mac or other), please inform about it in advance in the transaction description. Just remember that it takes up a lot more memory on your computer and it takes 1-2 days (from the moment the payment is received).

Important

Notify when the software has been successfully installed as the link to it is only temporary.

Installation as a PWA using Google Chrome for desktops

Step 1. Open the link you received by email.
Step 2. Press the installation button
Step 3. Notify when the software has been successfully installed as the link to it is only temporary.

Installation as a PWA using Google Chrome for mobile devices

Step 1. Open the link you received by email.
Step 2. Press the menu sidebar open button (marked in red).
Step 3. Then, press "Install app" item.
Step 4. Notify when the software has been successfully installed as the link to it is only temporary.

Reinstallation (works for the most browsers)

Step 1. Uninstall previous software versions
Step 2. Clear browser history and cache
Step 3. Install the current version of the software
